波の干渉とは？
2つの波が重なると、山と山が合えば強め合い、山と谷が合えば打ち消し合う。
下のスライダーで戻り波の位相（ずれ）を変えて、合計がどう変わるか見てみよう。

★ 完全に強め合っている — 共鳴！(ブースト)
バイクワッドとの関係:
バイクワッドの a1 は「戻り波の位相ずれ」を決める → どの周波数で共鳴するか
バイクワッドの a2 は「フィードバックの強さ」を決める → 共鳴の鋭さ(Q)

位相ずれ0°（山と山が一致）の周波数だけが強まる = EQ的な周波数選別の正体
