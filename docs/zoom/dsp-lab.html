体験ラボ

見るだけでなく、耳と手で effect の本質を確かめる。

このページは in-browser の簡易シミュレータです。入力音は短い test tone や pluck を内部生成し、dry / processed を聴き比べできます。DAW ほど本格的ではありませんが、DSP の仕組みを掴むには十分です。

コースハブへ戻る 上級編へ 解析一覧へ

4. Cabinet / FIR

簡易 IR を切り替えて、speaker / mic の色が高域と中域をどう変えるかを確認します。ここでは短い FIR として実装しています。

ラボの使い方

  1. まず dry を聴いて基準を作る
  2. 次に processed を聴き、グラフと一緒に何が変わったか確認する
  3. そのあと index.html の effect 解説へ戻ると、係数や DSP load の意味がかなり読みやすくなる
