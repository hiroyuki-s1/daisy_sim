上級分冊 Drive

Drive は「どんな非線形を、どこに、どれだけ入れるか」の設計問題です。

同じ gain でも、soft clipping、hard clipping、asymmetric clipping、dynamic drive では倍音の出方が大きく変わります。drive 系を読むときは回路名よりも、nonlinearity と pre/post filter の位置を見た方が本質が見えます。

上級ハブへ戻る Amp 編 体験ラボ
Nonlinearity

drive の個性は waveshaper の形でかなり決まります。

soft clipping

滑らかに飽和して、比較的自然な倍音増加を作ります。tube 的な質感の近似に使われやすいです。

hard clipping

急に切るので高次倍音が強く、aliasing にも厳しいです。aggressive な音になりやすいです。

asymmetric clipping

正負で違う歪み方をさせると偶数次倍音が増え、質感が変わります。

Pre And Post EQ

歪みの前後の filter が、歪みそのものと同じくらい重要です。

前段 filter

低域を削ってから歪ませると、音像が締まりやすくなります。高域を削ればザラつきが減り、中域を押せば抜ける drive になります。

filtered = preFilter(x) shaped = waveshape(filtered)

後段 filter

歪みで増えた高調波を整理し、最終キャラクタを決めます。tone ノブの多くは post-filter 側に近い役割を持ちます。

同じ waveshaper でも pre/post EQ を入れ替えるだけで別物に聞こえます。
Dynamic Behavior

上級 drive は入力レベルによって歪み方が変わることがあります。

dynamic drive

包絡やピッキングに応じて gain や filter を変え、弾き方への追従感を作ります。Comp の envelope の知識がここで再利用できます。

bias / sag 的表現

単純な static waveshaper ではなく、遅い成分を持たせて少しずつ歪み方が変わるようにすることで、アンプっぽい feel を作ることがあります。
