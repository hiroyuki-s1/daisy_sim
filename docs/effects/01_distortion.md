# 歪み系エフェクター設計教科書
## Distortion / Overdrive / Fuzz

---

## 1. 歪みの本質

歪みとは、**信号の振幅が一定の閾値を超えたとき波形が変形される現象**。
変形の仕方（ソフト/ハード、対称/非対称）と、どの周波数帯を歪ませるかが音の個性を決める。

### 生成される高調波

| クリッピングの種類 | 生成される高調波 | 音の印象 |
|-----------------|----------------|---------|
| 対称ソフトクリップ | 奇数次 (3, 5, 7...) | ブライト、パワフル |
| 対称ハードクリップ | 奇数次（強い）| 攻撃的、荒い |
| 非対称クリップ | 奇数次 + **偶数次** (2, 4, 6...) | 温かい、管球的 |
| 真空管の歪み | 主に偶数次 | 自然、音楽的 |

> **偶数次高調波 = 温かみの源泉**。真空管やゲルマニウムが好まれる理由。

## 2. アナログ回路の設計

### 2.1 クリッピング方式

#### ソフトクリップ（帰還型）— Tube Screamer方式

```
        Rf (+ Diodes)
       ┌──/\/\──┤>├──┐
       │               │
IN ─── OpAmp(-) ─────OUT
       │               │
       └─── R2 ────────┘
```

- ダイオードをオペアンプの**帰還ループ内**に入れる
- 信号が大きくなるほど帰還量が増え、ゲインが下がる → **動的なゲイン圧縮**
- ピッキングの強弱に自然に反応する「タッチレスポンス」

#### ハードクリップ（シャント型）— RAT / DS-1方式

```
        ゲイン段
IN ─── [×100] ─── [Diodes to GND] ─── OUT
```

- ゲイン段で増幅した後、グラウンドへのダイオードで固定閾値クリップ
- より攻撃的、レシオが信号振幅に依存しない

### 2.2 クリッピングダイオードの選択

| ダイオード種別 | 順方向電圧 Vf | ソフトネス | 音のキャラクター |
|-------------|-------------|----------|--------------|
| ゲルマニウム (1N34A, AA112) | 0.25〜0.35V | 最もソフト | 温かい、コンプ強め、ヴィンテージ感 |
| シリコン (1N4148, 1N914) | 0.6〜0.7V | ハード | クリア、精密、現代的 |
| ショットキー (BAT41) | 0.2〜0.4V | 中間 | 高速、ゲルマとシリコンの中間 |
| LED (赤) | 1.6〜1.8V | ソフト | 開放的、ダイナミクス豊か、明るい |
| LED (青/白) | 2.8〜3.5V | さらにソフト | 最もクリーン寄り |
| MOSFET体ダイオード | ~0.6V | 独特のサグ | 管球的コンプレッション |

#### 非対称クリッピング

順方向と逆方向で異なるダイオードを組み合わせ → 偶数次高調波を生成：

```
例: 順方向 = シリコン1個 (Vf 0.7V)
    逆方向 = シリコン2個直列 (Vf 1.4V)
→ クリップ閾値が非対称 → 波形の上下が異なる歪み方 → 2次高調波
```

### 2.3 Tube Screamer の核心（最重要）

TS-808 の帰還ループ RC 定数：

```
帰還ループ内: Cf = 0.047µF, Rf = 4.7kΩ
→ ハイパスカットオフ fc = 1/(2π × 4.7k × 0.047µ) ≈ 720 Hz

つまり：
- 720Hz以下 → 帰還が大きい → ゲイン低い → ほぼ歪まない
- 720Hz以上 → 帰還が小さい → ゲイン高い → しっかり歪む
```

さらに出力後の LPF：
```
Cout = 0.22µF, Rout = 1kΩ
→ ローパスカットオフ fc ≈ 720 Hz
→ クリッピングで生じた高域のざらつきを抑制
```

**結果**: 中域が強調された「ミッドハンプ」特性 → バンドで音が抜ける
**重要**: 歪みの**後**にEQを足すだけでは再現不可。フィルターが歪みの**中**に組み込まれている。

### 2.4 Fuzz Face の特殊性

```
入力インピーダンス ≈ 10kΩ（非常に低い）
```

- ピックアップのインダクタンス (数H) と入力インピーダンスが **LCローパスフィルター**を形成
- 高域が歪む**前**にロールオフ → 「スクイッシュ」した柔らかい歪み
- **バッファより必ず前に置く**: バッファ後だとこのピックアップとの相互作用が失われ、痩せた音になる

最適トランジスタバイアス：
```
Q1: β = 70〜80,  コレクタ電圧 = -0.5〜-0.7V
Q2: β = 110〜130, コレクタ電圧 = -4.5V (電源の半分)
```

### 2.5 JFET による管球エミュレーション

JFET をオーム領域でバイアスすると可変抵抗として動作し、三極管に近い特性：
- 偶数次高調波が豊富
- ゲートソース電圧 Vgs で歪み量を制御
- ドレインソース間の電位差が小さいほどソフトな歪み

## 3. DSP実装

### 3.1 処理チェーン

```
入力信号
  ↓
[プリEQ / LPF] ← TS808はここでHPFして低域を保護
  ↓
[アップサンプリング × 4〜8]  ← エイリアシング防止に必須
  ↓
[ウェーブシェーパー f(x)]
  ↓
[ダウンサンプリング]
  ↓
[ポストEQ]
  ↓
出力
```

### 3.2 ウェーブシェーパー関数

#### tanh（最もポピュラーなソフトクリップ）

```cpp
float waveshape_tanh(float x, float drive) {
    return tanhf(drive * x);
}
```

- 滑らかなソフトクリップ、数学的に安定
- drive を上げるほど激しい歪み
- 対称 → 奇数次高調波

#### atan（tanhの代替）

```cpp
float waveshape_atan(float x, float drive) {
    return (2.0f / M_PI) * atanf(drive * x);
}
```

- tanhより出力のスペクトル分布が若干異なる
- どちらが好みかは好みによる

#### 3次多項式ソフトクリップ

```cpp
float waveshape_cubic(float x) {
    x = fmaxf(-1.0f, fminf(1.0f, x));
    return x - (x * x * x) / 3.0f;
}
```

- 奇数次のみ（3次）→ よりシンプルなハーモニクス
- ハードリミット後なのでエイリアシングが少ない

#### ハードクリップ

```cpp
float waveshape_hard(float x, float drive) {
    x *= drive;
    return fmaxf(-1.0f, fminf(1.0f, x));
}
```

- シンプルだが**エイリアシングが最も激しい**
- オーバーサンプリングが特に重要

#### 非対称（偶数次高調波を加える）

```cpp
float waveshape_asymmetric(float x, float drive) {
    x = x * drive + 0.1f;  // DCオフセットで非対称化
    return tanhf(x) - 0.1f; // DC成分を除去
}
```

### 3.3 オーバーサンプリング（必須）

非線形処理はNyquist周波数を超える高調波を生成 → 折り返しエイリアシングが発生。

```
48kHz × 8× = 384kHz でウェーブシェーパーを動作させる

入力(48kHz) → [アップサンプリングフィルター] → (384kHz) → [ウェーブシェーパー]
                                                              ↓
出力(48kHz) ← [ダウンサンプリングフィルター] ← (384kHz)
```

| オーバーサンプリング倍率 | エイリアシング抑制 | CPU負荷 |
|--------------------|----------------|--------|
| 2× | 不十分 | ×2 |
| 4× | 許容範囲 | ×4 |
| **8×** | **推奨** | ×8 |
| 16× | 完璧だが重い | ×16 |

#### 代替: ADAA（Antiderivative Antialiasing）

オーバーサンプリングなしでエイリアシングを抑制：

```cpp
// tanh の不定積分 = ln(cosh(x))
float adaa_tanh(float x0, float x1) {
    if (fabsf(x1 - x0) < 1e-5f) return tanhf(0.5f * (x0 + x1));
    return (logf(coshf(x1)) - logf(coshf(x0))) / (x1 - x0);
}
```

計算コストがオーバーサンプリングより低い場合がある → 組み込みに有効。

### 3.4 周波数選択的歪み（Tube Screamer の DSP 再現）

```cpp
class TubeScreamerDSP {
    Biquad pre_hpf_;   // 歪み前 HPF (fc≈720Hz) → 低域を保護
    Biquad post_lpf_;  // 歪み後 LPF (fc≈720Hz) → 高域のざらつきを除去

    float Process(float in) {
        float filtered = pre_hpf_.Process(in);   // 低域カット
        float driven   = waveshape(filtered * drive_);  // 歪み
        return post_lpf_.Process(driven);        // 高域カット
    }
};
```

**NG（後からEQするだけ）**:
```cpp
float bad = waveshape(in * drive);  // フルバンドで歪み
return mid_boost_eq(bad);           // 後からEQ → 音が違う
```

### 3.5 多段ゲイン構成

1段で大きく歪ませるより、複数段に分けると有機的でダイナミクスに富む：

```cpp
// 2段構成の例
float stage1 = tanhf(in * 4.0f);          // 初段: 穏やかな歪み
float stage2 = tanhf(stage1 * drive_);    // 2段目: さらに歪み
// 各段のキャラクターが重なり合って複雑な音になる
```

## 4. アナログ PCB 設計のコツ

### ノイズ対策

| 項目 | 推奨 |
|------|-----|
| 接地方式 | **スター接地**（グランドは1点に集中） |
| デカップリング | ICの電源ピン直近に 100nF セラミック + 47µF 電解を並列 |
| 抵抗 | 信号経路には**金属皮膜抵抗**（炭素合成は1/fノイズが大きい）|
| 入力トレース | 短く、電源ラインから離す（ゲインが高いほどアンテナになりやすい）|
| 金属筐体 | 必ずグラウンドに接続（ファラデーケージになる） |

### 電源設計

```
リバースポラリティ保護: 電源ジャックと+レールの間に 1N4001 を直列挿入
デカップリング: 100Ω + 47µF でRC LPFを構成 → スイッチングノイズを除去
ポップ防止: バイパススイッチとレールの間に抵抗でチャージを逃がす
```

### ボルテージサグ

電源電圧を意図的に低下させる（容量を小さくする、直列抵抗）→
ヘッドルームが下がり動的なコンプレッションが生まれる → 「プッシュされた管球アンプ」の感触。

## 5. よくある失敗と対策

| 失敗 | 原因 | 対策 |
|------|------|------|
| 「デジタルっぽい」歪み | オーバーサンプリングなし | 最低4×、推奨8× |
| 後EQでは雰囲気が出ない | フィルターが歪みの外にある | フィルターを非線形処理の前後に配置 |
| ダイナミクスがない | 1段大ゲイン | 多段構成、ソフトクリップ優先 |
| ヒスノイズが大きい | 炭素合成抵抗、グラウンドループ | 金属皮膜抵抗、スター接地 |
| Fuzzが痩せる | バッファ後に接続 | チェーンの最初に置く |
| 温かみがない | 対称クリップのみ | 非対称クリップで偶数次高調波を追加 |
