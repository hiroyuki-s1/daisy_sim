# HPS (モジュレーション)

## 基本情報

| 項目 | 値 |
|---|---|
| ファイル | `HPS.ZD2` |
| エフェクト名 | HPS |
| ID | 0x060000d0 |
| バージョン | v1.50 |
| DSP負荷 | 6.64% |
| カテゴリ | モジュレーション |
| 説明 | This intelligent pitch shifter outputs the effect sound with the pitch shifted according to scale and key settings.

## パラメータ

| パラメータ | 範囲 | デフォルト | 説明 |
|---|---|---|---|
| Scale | 0-9 | 6 | Sets the pitch of the pitch-shifted sound added to the original sound. |
| Key | 0-11 | 0 | Sets the tonic (root) of the scale used for pitch shifting. |
| Tone | 0-10 | 6 | Adjusts the tone. |
| Mix | 0-100 | 70 | Adjusts the amount of effected sound that is mixed with the original sound. |
